## Amazon Elastic Container Registry (Amazon ECR)
Amazon Elastic Container Registry (Amazon ECR) is a managed Docker container registry. You push your container images to Amazon ECR, and you can then pull those images to launch containers. With Amazon ECR, you can compress, encrypt, and control access to your container images. You also manage versioning and and image tags. An Amazon ECR private registry is provided to each AWS account.

### Private
Private registries come with enterprise options to meet an organization’s security and compliance requirements, for example, if an organization’s software is proprietary and not allowed for public distribution.

### Public
A public registry commonly provides public repositories for container images. The primary purpose for public repositories is to share container images with anyone.
