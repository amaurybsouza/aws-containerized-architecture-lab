## 📖 About

This repository contains my personal notes, architectural diagrams, and hands-on labs from the **AWS Solutions Architect – Designing a Containerized Architecture** course.

The primary goal is to understand how to design scalable, secure, resilient, and cost-effective containerized applications using AWS services and cloud architecture best practices.

---

## 🎯 Learning Objectives

- Understand container fundamentals
- Design container-based architectures on AWS
- Deploy applications using Amazon ECS and Amazon EKS
- Build secure networking for containers
- Implement high availability and fault tolerance
- Design scalable architectures
- Secure workloads following AWS best practices
- Monitor containerized applications
- Optimize infrastructure costs
